# Presto - E-commerce Web Application

Presto is a full-stack web application developed during the Aulab Hackademy program.  
It is a marketplace platform where users can publish, browse, and manage ads.

## ✨ Features

- 🔐 User authentication (Laravel Fortify)
- 🔑 Password reset and recovery system
- 🛒 Create, edit and manage ads
- 📸 Multiple image upload with preview
- ✂️ Automatic image resizing (async jobs)
- 🧠 AI-based image processing:
  - Face detection and censorship
- 🔍 Full-text search (Laravel Scout + TNTSearch)
- 🌍 Multi-language support (IT / EN)
- 👨‍⚖️ Revisor dashboard:
  - Accept / reject ads
  - Role management via Artisan command
- 📬 "Work with us" form for revisor requests

## 🧠 Technical Overview

The application follows MVC architecture using Laravel framework and includes asynchronous jobs, RESTful logic, and database relationships.

## ⚙️ Tech Stack

- PHP / Laravel
- MySQL
- JavaScript
- Bootstrap
- Livewire
- Laravel Fortify
- Laravel Scout (TNTSearch)
- Google Cloud Vision API
- Spatie Image
- Git / GitHub
